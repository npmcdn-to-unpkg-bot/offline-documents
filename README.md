# offline-documents
Generate offline documents for Github projects, etc.

## Getting started

```
sudo easy_install pip
pip install grip
mkdir -p ~/source/_libs_
cd ~/source/_libs_
git clone https://github.com/request/request.git
cd ~/source
git clone https://github.com/xxd3vin/offline-documents.git
```

## Add new projects

```
cd ~/source/_libs_
git clone https://github.com/request/request.git
```
